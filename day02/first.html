Introduction to HTML

HTML (Hypertext Markup Language) is used to create documents on the World Wide Web.It is simply a collection of certain keywords called "Tags" that are helpful in writing the document to be displayed using a browser on the internet.HTML was developed by Tim Berners Lee in 1992.
